# Cài package va SUI CLI
1. SUI CLI (https://docs.sui.io/guides/developer/getting-started)
- Cài Homebrew hoặc Chocolatey
- Sau khi cài gõ sui --version để kiểm tra 

2. Cài package và các module
cd public
npm install

3. Sau khi cài kiểm tra
npm list
